# Pollution-Control-by-Identifying-Potential-Land-for-Afforestation-using-Computer-Vision
Air pollution has emerged as one of the most critical environmental challenges of the 21st century, especially in rapidly urbanizing metropolitan regions such as Mumbai, India. Increasing vehicular emissions, industrial activities, and unplanned urban expansion have significantly reduced green cover, leading to deteriorating air quality and adverse health impacts. Afforestation, defined as the large-scale planting of trees, is widely recognized as a sustainable and cost-effective solution for mitigating air pollution, as trees absorb carbon dioxide (CO₂), release oxygen, regulate temperature, and enhance ecological balance. However, identifying suitable land for afforestation in dense urban environments remains a complex task that traditionally requires extensive manual surveys, time, and resources.
This project presents an automated and scalable system that leverages Computer Vision techniques and freely available satellite imagery to identify potential land areas suitable for afforestation. The system accepts geographic coordinates (latitude and longitude) as input and retrieves high-resolution satellite tiles from the ESRI World Imagery service without requiring any API key or paid subscription. These tiles are programmatically stitched together to form a composite image representing the target region.
The core of the system utilizes OpenCV-based image processing techniques, specifically HSV (Hue, Saturation, Value) colour space transformation, to accurately detect vegetation and green cover within the satellite image. By applying carefully defined threshold ranges, the system isolates green regions and generates a binary mask highlighting vegetated areas. Further processing is performed to remove noise and improve detection accuracy.
Based on the detected green regions, the system computes the green coverage ratio and estimates the total land area using pixel-to-ground resolution conversion techniques. Using predefined assumptions regarding tree spacing and planting density, the system then approximates the number of trees that can potentially be planted in the identified regions. The results are presented in both numerical and visual formats, including a three-panel comparison chart generated using Matplotlib, showcasing the original satellite image, detected vegetation, and analytical output.
The entire pipeline is implemented in Python using open-source libraries such as NumPy for numerical computations, OpenCV for image processing, Pillow for image handling, Requests for data retrieval, and Matplotlib for visualization. The system is designed to be user-friendly, cost-effective, and easily deployable without reliance on proprietary software or paid APIs.
This project demonstrates the practical application of computer vision in addressing real-world environmental challenges. It provides a foundation for data-driven decision-making in urban planning, pollution control, and sustainable development. Furthermore, the system can be extended with advanced techniques such as NDVI-based analysis, machine learning-based land classification, and integration with Geographic Information Systems (GIS) for enhanced accuracy and scalability.
